Shortcuts:
Add custom shortcuts. Examples: shortcut:domain.zone (without search param); shortcut:domain.zone/?search_param= (with search param)

Please remember that the following commands only work on pages that were loaded using one of the http://, https:// or file:// protocols.

Keyboard:
Ctrl + Enter Show search
Esc Close search; Close tabs
Ctrl + ArrowDown Show tabs; Close Tabs
Ctrl + ArrowUp Show tabs; Close Tabs
Enter Execute query in search
F Focus caret in the search
Delete Remove suggest; Close tab
ArrowUp Move up in suggests; Move up in tabs
ArrowDown Move down in suggests; Move down in tabs
Touch:
Double touch (quickly) Show search; Close search; Close tabs
Triple touch (synchronously) Show tabs
